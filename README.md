## 项目目录结构

### Assets
项目文件夹
- Animation：动画相关文件
- Meshes：模型网格文件
- Scenes：游戏场景文件
- Scripts：脚本文件
  - UIFramework：UI框架
    - Core：框架核心代码，管理所有UI界面
    - Panel：面板类UI控制代码
    - Window：窗口类UI控制代码
- Settings：渲染管线设置
- Shader：着色器相关文件
- InputSystem_Actions.inputactions：输入配置文件

### Docs
开发文档文件夹
- 涉众分析：存放各版本涉众分析文档
- 特性分析：存放各版本特性分析文档
- 问题陈述：存放各版本问题陈述报告
- 愿景文档：存放各版本愿景文档

### Packages
Unity内置包管理器

### ProjectSettings
项目全局设置
